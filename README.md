# Password Manager 🔒  

A secure and user-friendly password management application built with Python, Tkinter, and Cryptography. This project ensures safe storage and retrieval of passwords by encrypting user credentials.  

## Features 🚀  
- **Secure Password Storage**: Encrypts passwords before saving them for maximum security.  
- **Decryption on Demand**: Retrieves and decrypts passwords only when needed.  
- **User-Friendly GUI**: Developed with Tkinter for an intuitive and simple interface.  
- **Random Password Generator**: Creates strong, random passwords for enhanced security.  
- **Search Functionality**: Easily locate saved credentials with a search feature.  

## Tech Stack 🛠️  
- **Python**: Core programming language.  
- **Tkinter**: GUI development.  
- **Cryptography**: Secure encryption and decryption of user data.
